# Database Design

## 1. User Collection

### Purpose

Stores user account information, academic details, career goals, preferences, and profile completion status.

### Sections

#### Basic Information
- Name
- Email
- Password (Encrypted)
- Avatar
- Profile Photo (Optional)

#### Academic Information
- College
- Branch
- Graduation Year
- Current Semester
- CGPA
- Active Backlogs

#### Career Goals
- Target Companies
- Preferred Role
- Target Package

#### Preferences
- Daily Study Hours
- Preferred Language
- Theme (Light / Dark)

#### Profile Status
- Profile Completion Percentage
- Created At
- Updated At
